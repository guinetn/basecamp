## git show

$id: commit, branch, tagname

git show $id
git show $id:$file

View the changes made in the commit
Show various types of objects
To view expanded details on Git objects. Git-show works on blobs, trees, tags, and commits. It even has a "pretty" option to massage the output to your liking.
https://www.atlassian.com/git/tutorials/git-show
