### Press "p" key

Progress bar like [Rabbit](https://rabbit-shocker.org/) displayed.

----

### Press "f" key

Full screen.

----

### Load your file

file:///path/to/your/sliderify/index.html?slide=your_slide

----

### URL Parameters

Key      | Value                  | Default
---------|------------------------|--------
slide    | directory name         | sample
progress | show progress bar      | false
limit    | progress limit minutes | 5
time     | progress start minutes | 0

----

### Custom CSS

write _custom.css_

```
 #p8 {
   background-image: url(background-image.jpg);
   background-size: cover;
   color: darkred;
 }
````
