## Transaction / Position

		ln order to complete a transaction, there must be a purchase and a sale. 
		Remember: realization of the profit or the loss only takes place once the transaction is completed. 
		The position opening is the transaction opening. 
		An open position is a transaction which has not yet been completed. 
		A closed position is a transaction which has been completed.
