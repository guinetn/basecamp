# LSP 

Language Server Protocol

Language Server: provide the language-specific smarts and communicate with development tools over a protocol that enables inter-process communication.
Standardize the protocol for how such servers and development tools communicate so a single Language Server can be re-used in multiple development tools (reduce effort)
